AKTA - Pro Study Tracker & Streak

AKTA is a modern and premium study tracking web application. It utilizes the Pomodoro technique and visual statistics to help you enhance your study focus. It is designed to work seamlessly in both local PC mode and cloud sync (Firebase) mode.

Follow these steps to run the tracker on your computer:
Create index.html: Copy the code from this project and save it as a file named index.html.
Firebase Connection (Optional but Recommended):

Paste your configuration details into the userFirebaseConfig object inside your index.html file.
Open in Browser: Double-click the file or drag-and-drop it into a Chrome or Edge browser.
